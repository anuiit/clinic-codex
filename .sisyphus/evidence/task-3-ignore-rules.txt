Patterns added to root .gitignore and reasons (2026-05-04):

OS / archive artifacts:
- .DS_Store                # macOS Finder metadata
- __MACOSX/                # macOS archive metadata directories
- ._*                     # AppleDouble sidecar files (._*) from extractions

Python:
- __pycache__/             # Python bytecode cache directories
- *.py[cod]                # Compiled Python bytecode files (.pyc, .pyo, .pyd)
- *.pyo                   # Older pyo bytecode
- venv/ .venv/ env/       # Virtual environment folders created by venv/virtualenv

Node / JavaScript:
- node_modules/            # npm/yarn installed packages
- npm-debug.log* / yarn-*.log*  # npm/yarn log files

Build outputs:
- dist/                    # distribution build output (frontend build)
- build/                   # alternative build output
- *.egg-info/              # Python packaging metadata directories

Editor / IDE / runtime:
- .vscode/ .idea/          # VSCode and JetBrains IDE project settings
- *.swp, *.swo             # vim swap files
- *.log                    # runtime log files
- coverage/                # test coverage outputs

Misc:
- Thumbs.db                # Windows Explorer thumbnail cache

Safety / exceptions recorded separately in task-3-ignore-safety.txt
