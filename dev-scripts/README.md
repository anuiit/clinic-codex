# dev-scripts/

Scripts used during development for one-off patches and fixes. Not part of the main application.

## Contents

- `patch.js` — One-off patch script (historical, do not run in production)
- `patch2.js` — One-off patch script v2 (historical, do not run in production)
- `patch_workspace.js` — Workspace patch script (historical)
- `patch_workspace.sh` — Workspace patch shell script (historical)

These scripts were used during early development to apply fixes to the codebase.
They are kept here for reference but should not be needed for normal development.
